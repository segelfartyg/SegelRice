# SegelRice - Mountain

A complete Arch Linux rice featuring a Hyprland-based Wayland desktop environment with a mountain-themed aesthetic.

## About

This repository contains the configuration files and package list for a custom Arch Linux setup centered around Hyprland, a dynamic tiling Wayland compositor. The "Mountain" theme provides a cohesive visual experience across all components.

## Components

- **Window Manager**: Hyprland (Wayland compositor)
- **Status Bar**: Waybar
- **Terminal**: Kitty
- **Application Launcher**: Rofi
- **Display Manager**: SDDM with custom Mountain theme
- **Wallpaper Manager**: Hyprpaper
- **Shell**: Custom shell configuration

## Included Configurations

The `configs/mountain/` directory contains ready-to-use configuration files for:
- Hyprland window manager settings
- Waybar status bar styling and modules
- Kitty terminal appearance
- Rofi launcher theming
- SDDM login screen with custom Mountain theme
- Hyprpaper wallpaper setup
- Shell environment
